Fundamentos de CSS

CSS es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc. (MDN contributors, 2022)

MDN contributors. (2022). ¿Qué es el CSS? - aprende sobre desarrollo web: MDN. Aprende sobre desarrollo web | MDN. Retrieved from https://developer.mozilla.org/es/docs/Learn/CSS/First_steps/What_is_CSS

Frameworks CSS

Un framework es un marco o esquema de trabajo generalmente utilizado por programadores para realizar el desarrollo de software. Utilizar un framework permite agilizar los procesos de desarrollo ya que evita tener que escribir código de forma repetitiva, asegura unas buenas prácticas y la consistencia del código. (Arimetrics , 2022)

Arimetrics . (2022). Qué es framework - Definición, Significado y Ejemplos. Arimetrics. Retrieved from https://www.arimetrics.com/glosario-digital/framework
